# 運用方針とマネタイズ戦略

## 基本方針

このアカウント群は、アフィリエイトリンクそのものがブランドになる設計にします。

軸は以下です。

```text
要らないのに欲しくなる
くだらないけど使える
無駄なのに日常で使える
見た瞬間に誰かに見せたくなる
```

最初から売り込みを強めず、投稿の信頼を「商品選定の面白さ」で作ります。

## フェーズ設計

### Phase 1: 安定稼働確認

期間:

```text
開始から7日
```

目的:

- X / Threadsの自動投稿が止まらないことを確認する
- 投稿頻度に対してアカウント制限が出ないことを確認する
- 投稿文の方向性を整える

投稿内容:

- おはよう、夕方、夜の軽い観察投稿
- 商品名なし
- リンクなし
- ハッシュタグなし

見る指標:

- 投稿成功数
- 失敗ログ
- インプレッション
- プロフィールアクセス
- 不自然な制限や警告の有無

### Phase 2: キャラ投稿

期間:

```text
2週目
```

目的:

- アカウントの文体を固める
- 商品なしでも読める投稿を増やす
- 「それ要ります？」の世界観を作る

投稿内容:

- 商品カテゴリの話題
- 無駄な便利グッズへの観察
- 日常の小ネタ
- まだリンクなし

例:

```text
便利グッズって、便利さより先に見た目で負けることがあります。
必要性を説明できないのに欲しいなら、もうそれは商品側の勝ちでいい気がします。
```

### Phase 3: 商品紹介テスト

期間:

```text
3週目以降
```

目的:

- Amazon / 楽天の商品を低頻度で紹介する
- クリック率を測る
- 売れやすい商品カテゴリを特定する

投稿比率:

```text
通常投稿 70%
商品紹介 30%
```

商品条件:

- 価格は1,000円から5,000円中心
- 見た目で説明できる
- ジョーク性がある
- 日常使用できる
- 危険物、医療効果、美容効果、成人向けは避ける
- レビュー、在庫、価格が極端に怪しくない

### Phase 4: 横展開

対象:

- Pinterest
- Instagram
- Threads
- X

形式:

- X: 短文ツッコミ + リンク
- Threads: 少し長めの観察文
- Pinterest: 商品画像/生成画像 + 保存されやすいタイトル
- Instagram: 正方形画像 + キャプション

Pinterest / Instagramは画像URLが必要なため、Cloudflare上に画像置き場を作ってから本格化します。

## マネタイズ導線

### Amazon

役割:

- 商品数が多い
- ネタ商品を探しやすい
- ユーザーが買いやすい

使い方:

- 低単価商品でクリックと購入の流れを作る
- 商品単体の収益より、関連購入の取りこぼしを狙う
- 最初はAmazon中心

### 楽天

役割:

- 楽天ユーザー向けの補完
- ポイント訴求が強い
- 同系統商品を別導線で紹介できる

使い方:

- Amazonで弱い商品を楽天で補完
- 楽天セール時期に寄せる
- Pinterestと相性が良い商品を優先する

### TikTok Shop

役割:

- 将来的な収益拡大枠
- 短尺動画と相性が良い

使い方:

- 初期最小構成には入れない
- X / Threads / Pinterest / Instagramで勝ち商品が見えた後に動画化する
- 売れ筋だけをTikTok Shop向けに変換する

## 収益を伸ばす考え方

最初に狙うのは、いきなり高単価商品を売ることではありません。

優先順位:

1. 投稿が止まらない
2. アカウントが制限されない
3. クリックされる商品カテゴリを見つける
4. 保存される画像フォーマットを見つける
5. クリック率の高い投稿文を残す
6. 勝ち商品だけリンク付き投稿を増やす

伸びる可能性が高い商品カテゴリ:

- パーティグッズ
- 子どもと大人が一緒に使える遊び道具
- キャンプ、BBQ、小型アウトドア用品
- デスク周りの謎便利グッズ
- キッチンの変な時短グッズ
- プレゼント向けジョーク商品
- 掃除、収納の見た目が面白い用品

避けるカテゴリ:

- 医療効果をうたう商品
- 美容効果を強くうたう商品
- サプリ
- 成人向け
- 武器性が強い商品
- 法規制や安全性の説明が難しい商品
- レビューが極端に怪しい商品

## 投稿比率

初期:

```text
通常投稿 100%
商品リンク 0%
```

商品紹介テスト:

```text
通常投稿 70%
商品紹介 30%
```

安定後:

```text
通常投稿 50%
商品紹介 40%
まとめ/再掲 10%
```

商品紹介が増えても、全投稿を広告にしません。アカウントの価値は「面白い商品を選ぶ目」に置きます。

## 週次確認

毎週見るもの:

- Xインプレッション
- Threads反応
- プロフィールアクセス
- クリック数
- Amazon / 楽天成果
- 失敗ログ
- 伸びた投稿文
- 伸びなかった投稿文

判断:

- インプレッションが伸びた文体はテンプレート化する
- クリック率が高いカテゴリを残す
- 反応が弱いカテゴリは一時停止する
- 成果が出るまで投稿量を急増させない
