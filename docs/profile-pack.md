# SNSプロフィール設定

アカウント名の推奨は `それ、要ります？研究所` です。本人名・有名人画像・公認に見える表現は使いません。

## 初期確認用プロフィール共通方針

初期確認用プロフィールでは、商品リンクやPR色を強く出しません。アカウントの方向性だけを伝え、投稿安定後にアフィリエイト表記を追加します。

- 「要らないのに欲しい」を明確にする
- 自動運用テスト中であることを過度に隠さない
- 商品リンクなし運用中であることを明記する
- 本人名、有名人名、公認風表現は使わない
- 本運用開始後に広告表記を追記する
