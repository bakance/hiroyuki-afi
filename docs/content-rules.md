# 投稿文章ルール

この文書は、初期ウォームアップ投稿の文章ルールです。

## 現在の運用段階

現在は、アフィリエイト前のウォームアップ段階です。

- 商品リンクを入れない
- 商品名を入れない
- ハッシュタグを入れない
- 煽り、炎上狙い、断定的な人格批判を入れない
- X / Threads は同じ方向性の短文で投稿する
- Pinterest / Instagram は画像URLを設定するまで自動投稿しない

## 投稿頻度

1日3回です。

```text
08:20 JST
17:40 JST
21:15 JST
```

## 文章トーン

初期1週間は、淡々とした観察口調にします。

- 断定しすぎない
- 攻撃的にしない
- 商品を売り込まない
- 「要らないのに気になる」というアカウントの方向性だけを薄く出す
- 文字数は短めにする

## 現在の投稿候補

重複投稿を避けるため、朝・夕方・夜それぞれ10本ずつの候補を用意します。ここでは抜粋だけ記載します。実際の候補全文は `scripts/warmup-content.mjs` を確認してください。

朝の抜粋:

```text
おはようございます。今日も、なくても困らないものを冷静に眺めていきます。
おはようございます。便利と無駄の境界線、だいたい朝は少し曖昧です。
おはようございます。必要性はさておき、気になるものは気になります。
```

夕方の抜粋:

```text
夕方ですね。今日も、生活に必須ではないものほど妙に記憶に残ります。
おつかれさまです。使う理由より、欲しくなる理由のほうが先に来る日があります。
夕方の観察です。人はたまに、用途よりも雰囲気で物を欲しがります。
```

夜の抜粋:

```text
こんばんは。夜は判断力が少し甘くなるので、変なものが魅力的に見えます。
今日もおつかれさまでした。必要ないのに気になるもの、だいたい明日も気になります。
こんばんは。買う理由を探し始めた時点で、少し負けている気がします。
```

## SNS別の末尾文

X:

```text
今日も淡々といきます。
```

Threads:

```text
ゆるく観察していきます。
```

Pinterest:

```text
気になったものはボードに残していきます。
```

Instagram:

```text
淡々と記録していきます。
```

末尾文は日替わりで付く場合と付かない場合があります。

## 事前確認方法

方法1: 7日分をまとめて確認する

```text
GitHub > Actions > Warmup Draft Generator > Run workflow
```

実行後、`warmup-manual-posts` というArtifactを開きます。中の `manual-posts.md` に、7日分の投稿予定文が入っています。

方法2: 実投稿せず、次の投稿だけ確認する

```text
GitHub > Actions > Auto Warmup Post > Run workflow
dry_run = true
```

`dry_run=true` の場合、投稿はされません。ログに投稿予定文だけが表示されます。

方法3: 実投稿する

```text
GitHub > Actions > Auto Warmup Post > Run workflow
dry_run = false
```

実投稿後、ログに投稿URLまたは投稿IDが表示されます。

## 次フェーズの文章ルール

1週間のウォームアップが安定した後、アフィリエイトなしで「淡々ツッコミ口調」の投稿へ移行します。

その段階では以下を追加します。

- 商品カテゴリの話題を入れる
- まだ商品リンクは入れない
- 実在人物のなりすまし表現は避ける
- 本人名、本人画像、本人を名乗る表現は使わない
- 口調は「論理っぽく淡々とツッコむ」程度に留める

アフィリエイト投稿へ移行するのは、投稿安定とアカウント状態を確認してからです。

## 次フェーズの文体設計

実在人物の本人なりすまし、本人名、公認に見える表現は使いません。文体は、以下の要素を抽象化して使います。

- 論理っぽく始める
- 強く売り込まず、少し引いた位置から観察する
- 「それ必要ですか？」という問いを入れる
- 一度否定してから、結局ほしくなる理由を出す
- 断定ではなく「気がします」「じゃないですかね」を使う
- 誇大広告、効果保証、人格攻撃は避ける

文章構造:

```text
1. まず不要そうに見える点を言う
2. でも欲しくなる理由を一つだけ出す
3. 日常で使える場面を短く添える
4. 最後に淡々としたツッコミで落とす
```

例:

```text
水鉄砲にバズーカ感を求める必要、普通はないんですよね。
ただ、こういう無駄な迫力って、なぜか一回試したくなるじゃないですか。
庭とかキャンプで使うなら、たぶん子どもより大人が先に撃ちます。
```

```text
これ、生活必需品ではないです。
でも「なくても困らない」と「あると妙に楽しい」は、別の話なんですよね。
そういう物に弱い人、一定数います。
```

```text
便利グッズって、便利さより先に見た目で負けることがあります。
必要性を説明できないのに欲しいなら、もうそれは商品側の勝ちでいい気がします。
```

NG:

```text
本人を名乗る
本人名を使う
公認に見える表現を使う
誰かを馬鹿にする
絶対に売れる、必ず効果がある、などの断定をする
```
